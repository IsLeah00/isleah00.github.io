# IsLeah Portfolio Website

Personal portfolio website for showcasing my software development projects, professional experience, certifications, and technical skills.

## Overview

This portfolio presents selected personal, academic, and professional work, including:

- Software development projects
- Work experience highlights
- Certifications
- Technical skills and preferences
- Contact information

The website is built as a modern Angular application with multilingual support.

## Tech Stack

- Angular
- TypeScript
- SCSS
- Tailwind CSS
- Transloco for internationalization
- Angular Material
- Google Fonts

## Features

- Responsive portfolio layout
- Multilingual content support: English, Hungarian, and German
    - The initial language is selected automatically from the user's browser/system language settings, with English used as the fallback language.
- Project showcase with screenshots
- Certification gallery with image preview
- Contact dialog
- Custom styling and reusable components

## Project Structure

```text
